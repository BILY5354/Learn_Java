### 数组的使用

声明->创建->初始化

- 动态定义
- 

```java
//声明
int myArray[];
myArray[] int;//推荐


//创建:数据类型 数组名[] = new 数据类型[大小];
int myArray[] = new int[5];//动态定义方式1

int myArray[];//动态定义方式2
myArray = new int[5];

int myArrayLength = myArray.length;//获取数组长度，为5


//初始化
int a[] = {1,2,3,4,5};//静态初始化

int a[] = new int[5];//这两句相当于静态初始化
a[0] = 1;	a[1] = 2;	a[2] = 3;	
a[3] = 4;	a[4] = 5;
```
